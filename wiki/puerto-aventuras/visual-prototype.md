# Puerto Aventuras Super-App
## Visual Prototype & UI Mockups

---

## Design System

### Color Palette

```
Primary Colors:
┌─────────────────────────────────────────────────────┐
│  Ocean Blue:    #0066CC  (CTA buttons, links)        │
│  Sand White:    #FAF8F5  (Backgrounds)              │
│  Sunset Orange: #FF6B35  (Alerts, notifications)     │
│  Palm Green:    #22C55E  (Success states)           │
│  Deep Navy:     #0A1628  (Text, headers)            │
│  Coral Red:     #EF4444  (Errors, warnings)         │
└─────────────────────────────────────────────────────┘
```

### Typography

```
Headings:   Poppins Semibold (24px, 20px, 18px)
Body:       Inter Regular (16px, 14px)
Buttons:    Poppins Medium (16px, 14px)
Captions:   Inter Regular (12px)
```

### Components

```
Buttons:
┌──────────────────────┐  ┌──────────────────────┐
│   Primary Action     │  │   Secondary Action   │
│  (Ocean Blue bg)     │  │  (Outline only)       │
└──────────────────────┘  └──────────────────────┘

Cards:
┌────────────────────────────────────────┐
│                                        │
│   Content with shadow (elevation-2)    │
│   Rounded corners (8px)                │
│                                        │
└────────────────────────────────────────┘

Input Fields:
┌──────────────────────────────────┐
│ Email/Phone                      │  ← Placeholder text
└──────────────────────────────────┘
```
